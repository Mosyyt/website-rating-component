## What is Rating Component?

Rating Component is a self-contained interactive UI widget. The user selects a rating from 1 to 5, submits it, and sees a thank-you screen that reflects their choice. It was built as a Frontend Mentor-style challenge — one component, two states, clean transitions.

---

## Features

- **5-point rating selector** — click a number to select your rating
- **Submit button** — confirms the choice and transitions to the result screen
- **Thank-you screen** — displays the selected rating back to the user
- **State transitions** — smooth switch between rating and result views
- **Responsive card** — centered layout that works on any screen size

---

## How to Use

```bash
start index.html      # Windows
open index.html       # macOS
```

Click a number, press **Submit**, see the result.
